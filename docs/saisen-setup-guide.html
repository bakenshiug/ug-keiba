第 一 章

お賽銭文化と神宮の方針

なぜお賽銭か（投げ銭ではなく）

「投げ銭」「サポート」「チップ」と言わず、「お賽銭」 と呼ぶのが神宮ブランド。

  • 世界観統一：神宮の参拝者という体験全体に組み込まれる
  • 気軽さ：「払う」じゃなく「投げる／納める」精神
  • ご利益思想：見返り（的中）を期待する文化と相性◎

神宮的お賽銭の3原則

  1. 強制しない：サイトは無料で動かす。賽銭は「気持ち」のみ
  2. ネタとして楽しむ：「神獣の修行料」「木材メンテ費」等
  3. 透明性：賽銭が何に使われるか、毎週報告

💡 神宮的トーン

「お願いします」ではなく、「神宮にご納得頂けたら、御布施を」のスタンス。 押し売り感ゼロで、参拝者に判断を委ねる文化。

第 三 章

OFUSE セットアップ（推奨・本命）

OFUSEは日本のクリエイター御布施プラットフォーム。 名前が「御布施」で神宮ブランドと完璧マッチ。

登録手順（5分）

  1. https://ofuse.me/ にアクセス
  2. 「クリエイターとして登録」クリック
  3. メアド・パスワード設定
  4. クリエイター名：「UG競馬神宮 ／ 神主ニック」
  5. プロフィール：神宮の説明＋note へのリンク
  6. 振込口座登録（個人銀行口座OK）
  7. マイページのURLが発行される（例：https://ofuse.me/ug-jingu）

プロフィール例文

UG競馬神宮、神主ニックです。
AI神獣ギーニョと共に、言霊×外厩で全レース予想を公開しております。

お賽銭は、神宮サーバー代・ギーニョの木材メンテナンス費・神獣たちの修行料に充てさせて頂きます。
ご納得頂けましたら、御布施を頂けますと幸いです。

⛩ 神宮：https://bakenshiug.github.io/ug-keiba/

手数料・振込

  • 受け取り額の 10% がOFUSE手数料
  • 月末締め・翌月15日振込
  • 1回 ¥1,000以上 の累計で振込実行
  • 確定申告時に「雑所得」として記載

⚠ 注意

クリエイター登録時、本人確認書類提出あり（運転免許証等）。 審査に2〜3日かかる場合あるので早めに登録。

第 四 章

PayPay 個人間送金（QRコード方式）

手数料 0%・即時受け取り。日本ユーザーには最速。

QRコード発行手順

  1. PayPay アプリを開く
  2. ホーム画面の 「送る・受け取る」 をタップ
  3. 「受け取る」 をタップ
  4. QRコードが表示される → スクショ保存
  5. このQRコードを画像として、サイトに掲載

サイト掲載イメージ

PayPay でお賽銭を投げる

[QRコード画像を表示]

⚠ 注意

  • 個人間送金は 本人確認済みアカウント 同士のみ
  • PayPayマネーライト（チャージ残高）からは送金可
  • QRコード画像は、定期的に更新するのが安全

💡 名前バレ対策

PayPayでは送金時に相手の本名が表示される場合あり。 プライバシー重視なら、別アカウント（ニックネーム登録）を推奨。

第 五 章

Amazon ほしいものリスト

物品で支援を受ける形式。手数料0%、配送はAmazon任せ。

作成手順

  1. Amazon アカウントにログイン
  2. 右上の「アカウント＆リスト」→「ほしい物リスト」
  3. 「リストを作成」
  4. 名前：「神宮ほしい物リスト」（公開設定）
  5. 送付先住所：受取人名「UG競馬神宮」匿名（コンビニ受取）も可
  6. 欲しいものを追加（例：マイク、書籍、Stripe API 本、競馬書籍、コーヒー豆、PCグッズ等）
  7. 「リストを共有」→ URLをコピー

推奨ラインナップ例

カテゴリ商品例賽銭金額目安
消耗品コーヒー豆・お茶¥500〜¥2,000
書籍競馬書籍・Python本¥1,000〜¥3,000
機材マイク・キーボード・モニター¥5,000〜
サーバーVPSギフトカード¥1,000〜
謎木材（ギーニョの体）ネタ

⚠ 住所バレ対策

「コンビニ受取」または「Amazon Hub ロッカー」受取を設定すれば、 自宅住所はバレません。神宮ブランドを守るためにも推奨。

第 六 章

Stripe Payment Links（自前決済・上級）

クレジットカード決済・Apple Pay対応。手数料3.6%。 将来 ¥3,000以上の「特別お賽銭」コース用。

セットアップ手順

  1. https://stripe.com/jp でアカウント作成
  2. 本人確認・銀行口座登録（審査3〜5日）
  3. ダッシュボード →「支払い」→「決済リンク」
  4. 「決済リンクを作成」
  5. 商品名：「神宮 御布施（金の賽銭）」
  6. 金額：固定 or「お客様が決定」を選択
  7. リンクが発行される（例：buy.stripe.com/xxx）

第 七 章

note サポート機能

note の有料記事下に表示される「クリエイターをサポート」機能。 既に note アカウントあるなら設定不要。

確認手順

  1. note にログイン
  2. マイページ → アカウント設定
  3. 「サポート機能」が ON になっているか確認
  4. 振込口座登録

記事下に自動で「クリエイターをサポート（¥100/¥500/¥1,000）」ボタンが表示される。

💡 連動戦略

note 記事内で「サイト本体ではOFUSEもあります」と誘導 → サイトでは「note サポートも歓迎」と記載。 お互いに送客し合う仕組み。

税務注意

⚠ 確定申告

  • 年間賽銭収入が ¥200,000超 → 確定申告必須（雑所得）
  • サラリーマン副業の場合、年間¥200,000以下なら申告不要（住民税のみ要申告）
  • OFUSE・Stripe は年間収支が記録される
  • PayPay 個人間送金も実態として収入なら課税対象
  • 領収書・記録を必ず保管

不正・トラブル対応

  • 賽銭返金リクエスト → 基本的に不可、ただし二重送金等は対応
  • 悪意ある大量送金 → サービス側に通報
  • QRコード偽造リスク → 定期更新で対応
