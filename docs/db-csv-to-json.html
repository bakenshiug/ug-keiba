💎 金塊DB 変換ツール

テキストをそのまま貼り付けるだけでJSON生成 — Claude Code不要・トークン消費ゼロ

📋 使い方

  1. コース名とファイル名を入力
  2. データをそのまま貼り付け（カンマ区切り or タブ区切り、どちらでもOK）
  3. 「変換」ボタンを押す → プレビューで確認
  4. 「ダウンロード」でJSONを保存
  5. docs/db/ フォルダに置く
  6. docs/db/index.json に1行追加（ページ下のテンプレをコピー）
  7. git add → git commit → git push
