💎 金塊DB CSV→JSON 変換ツール

Claude Code不要・ブラウザだけでJSONを生成できます

📋 使い方（トークン消費ゼロ手順）

  1. コース名・条件データを入力
  2. 「JSONを生成」ボタンを押す
  3. 「ダウンロード」で xxx-shiba-yyyy.json を保存
  4. docs/db/ フォルダに置く
  5. docs/db/index.json に1行追加（下記テンプレ参照）
  6. git add docs/db/xxx.json docs/db/index.json
  7. git commit -m "db: 〇〇コース金塊DB追加"
  8. git push
